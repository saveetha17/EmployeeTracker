# EmployeeTracker


## Description

EmployeeTracker is a command-line application that allows us to view and manage the departments, roles, and employees in an organization to sort the datas effectively.


## Technologies Used

* JavaScript
* Node.js
* NPM
* Inquirer
* Mysql2
